# Family Chat — описание функционала приложения

Документ описывает приложение **Family Chat** с точки зрения пользователя: что оно умеет, как выглядит, какие есть экраны, кнопки и действия. Технические детали реализации здесь не рассматриваются.

---

## 1. Что это за приложение

**Family Chat** — семейный мессенджер для близких родственников. В одном приложении собраны:

- общение в чатах (семейный, личные, групповые);
- аудиозвонки в личных чатах;
- лента семейных событий;
- список родственников и визуальное дерево семьи;
- общая и личная фотогалерея с альбомами;
- семейный календарь с днями рождения и событиями;
- профиль каждого участника.

Приложение рассчитано на одну «семью» — закрытое круг родственников, приглашённых по ссылке.

Доступно на **Android**, **iPhone** и в **браузере** (веб-версия и PWA на [familychat-app.ru](https://familychat-app.ru/app)).

## 2. Первый запуск и вход

### 2.1. Экран входа

При открытии приложения без активной сессии показывается экран входа с градиентным фоном и карточкой входа:

- логотип и заголовок **Family Chat**;
- подзаголовок «Семейный мессенджер»;
- краткое описание приложения;
- блок **«Войти в аккаунт»** с тремя иконками соцсетей в ряд:
  - **ВКонтакте**
  - **Яндекс**
  - **Google**

Тап по иконке запускает вход через выбранный сервис. Во время входа показывается индикатор загрузки.

Если регистрация нового аккаунта через Google недоступна (ограничение для пользователей из РФ), показывается предупреждение с просьбой выбрать другой способ входа (ВК или Яндекс). Существующие пользователи Google по-прежнему могут войти.

Под карточкой входа — ссылки:

- **Политика конфиденциальности**
- **Пользовательское соглашение**

Ссылки открывают соответствующие страницы на сайте приложения.

После успешного входа пользователь попадает либо в онбординг (если ещё не в семье), либо в основное приложение.

## 3. Онбординг (подключение к семье)

Показывается, если пользователь вошёл, но ещё не завершил настройку или не состоит в семье.

### Шаг 1 — выбор сценария

Экран **«Добро пожаловать»**:

- **Нет, создать свою семью** — пользователь создаёт новую семью;
- **Да, у меня есть приглашение** — пользователь присоединяется по ссылке.

Если приложение открыто по ссылке-приглашению, этот шаг может быть пропущен автоматически.

### Шаг 2 — профиль

Пользователь заполняет:

- **Имя**
- **Фамилия**
- **Пол** (Мужской / Женский)
- **День рождения** (выбор в календаре)
- галочка **Показывать год** в дне рождения

Кнопка **Продолжить**.

### Шаг 3а — создание семьи

Если пользователь создаёт семью:

- автоматически предлагается название вида «Семья {Фамилия}»;
- кнопка **Создать семью**.

### Шаг 3б — принятие приглашения

Если пользователь пришёл по приглашению — после профиля сразу начинается присоединение к существующей семье.

### Шаг 4 — приглашение первого родственника (опционально)

После создания семьи:

- выпадающий список **степени родства** (Супруг/а, Сын, Дочь, Брат, и т.д.);
- **Пригласить** — создаёт ссылку и открывает системное меню «Поделиться»;
- **Пропустить и войти в чат** — перейти в приложение без приглашения.

### Шаг 5 — уточняющие вопросы о родстве

Если при присоединении к семье связь неоднозначна (например, добавляется сын, нужно указать обоих родителей), приложение задаёт до нескольких вопросов с вариантами ответов. Кнопка **Завершить** завершает онбординг.

После онбординга открывается основной экран приложения.

## 4. Основная навигация

Внизу экрана всегда (кроме некоторых вложенных экранов из «Ещё») видна панель из четырёх вкладок:

| Вкладка | Название | Что открывает |
|---------|----------|---------------|
| 1 | **Главная** | Лента семейных событий |
| 2 | **Чат** | Список всех чатов (с бейджем непрочитанных) |
| 3 | **Семья** | Участники и дерево родства |
| 4 | **Ещё** | Меню: Галерея, Календарь, Профиль |

Вверху — строка заголовка с названием текущего раздела и контекстными кнопками (зависят от вкладки).

## 5. Вкладка «Главная» — лента

### 5.1. Что видит пользователь

**Лента** — хронологический поток событий в семье. Показываются только те события, к которым у пользователя есть доступ (см. раздел 12).

**Фильтр по людям** — горизонтальная полоса аватаров вверху:

- иконка «все» — показать события всех;
- аватар конкретного человека — показать только его события.

**Разделы ленты:**

- **Новые** — события, появившиеся с прошлого просмотра;
- **Просмотрено** — более ранние события.

Каждая карточка события содержит:

- аватар автора;
- заголовок события;
- подзаголовок «где это произошло» (чат, альбом, галерея, календарь);
- время;
- превью (текст сообщения, фото, лайки/комментарии);
- кнопку **Перейти** (стрелка) — открыть источник события.

Поддерживается прокрутка вниз (подгрузка старых событий) и обновление потягиванием вниз.

### 5.2. Типы событий в ленте

| Событие | Как выглядит | Куда ведёт «Перейти» |
|---------|--------------|----------------------|
| Сообщение в чате | «Иван написал(а) в чате» + название чата + текст | Соответствующий чат |
| Фото в галерею | «Иван добавил(а) фото» | Галерея |
| Фото в альбом | «Иван добавил(а) фото в альбом» + название альбома | Альбом |
| Лайк на фото | «Иван лайкнул(а) фото» | Просмотр фото |
| Комментарий к фото | «Иван прокомментировал(а) фото» | Просмотр фото |
| Новый участник | «Мария присоединилась к семье» | Профиль участника |
| Обновление профиля | «Иван обновил(а) профиль» + что изменилось | Профиль участника |
| Событие календаря | Название события | Календарь |

**Важно:** личные переписки (1-на-1) **никогда** не попадают в ленту. Сообщения из групповых чатов видны только участникам этой группы. Фото и альбомы — только тем, у кого есть к ним доступ.

### 5.3. Действия

- тап по карточке или **Перейти** — открыть источник;
- тап по миниатюре фото — открыть просмотр фото;
- тап по аватару в фильтре — отфильтровать ленту по человеку.

Лента автоматически отмечается просмотренной через короткое время после открытия новых событий.

## 6. Вкладка «Чат»

### 6.1. Список чатов (хаб чатов)

Заголовок: **Семейный чат**.

**Кнопки в шапке:**

- **Поиск** (лупа) — показать/скрыть поле поиска по названию чата;
- **Создать группу** (+) — создать новый групповой чат.

**Фильтры-чипы:**

- **Все** — все чаты;
- **Семья** — только общий семейный чат;
- **Личные** — только личные переписки 1-на-1;
- **Группы** — только групповые чаты.

**Список чатов.** Каждая строка:

- аватар (собеседника, группы, семейного чата или чата подготовки поздравления);
- название чата;
- превью последнего сообщения;
- время;
- бейдж с числом непрочитанных (если есть).

У **семейного чата** и **чата подготовки поздравления** вместо пользовательского фото показывается фирменная картинка чата.

Тап по строке — открыть чат.

Пустой список: «Нет чатов» или «Чаты не найдены» (при активном поиске).

### 6.2. Создание группового чата

Экран **«Новая группа»**:

- поле **Название группы**;
- список участников семьи с чекбоксами (себя выбрать нельзя);
- в шапке: **Выбрать все** / **Снять все**;
- кнопка **Создать** — создаёт группу и открывает чат.

### 6.3. Экран переписки

Открывается при выборе чата из списка, из ленты, из профиля участника, из уведомления или при пересылке/шаринге.

#### Шапка (обычный режим)

- **Аватар и название чата** — тап открывает информацию о чате;
  - в шапке показывается аватар собеседника, группы или фирменная картинка семейного/праздничного чата;
- **Подзаголовок:**
  - в личном чате — статус собеседника (Онлайн / Был только что / Был недавно / Был давно);
  - в группе/семейном — число участников;
- **Аудиозвонок** (только в личном чате) — начать звонок;
- **Поиск** — поиск по сообщениям в этом чате.

#### Шапка (режим выбора сообщений)

Появляется при долгом нажатии на сообщение:

- счётчик **N выбрано**;
- **Выбрать все** / **Снять все**;
- удалить выбранные;
- скопировать;
- закрыть режим выбора.

#### Область сообщений

- сообщения в хронологическом порядке (новые внизу);
- прокрутка вверх подгружает старые сообщения;
- системные сообщения (например, «Иван присоединился») — отдельными плашками;
- **история звонков** в личных чатах — отдельные плашки с типом звонка, длительностью и кнопкой **Перезвонить** (см. раздел 6.6);
- **отложенные сообщения** — показываются в ленте до момента отправки с иконкой часов и временем отправки;
- в групповых чатах у чужих сообщений — аватар и имя отправителя (тап по аватару → профиль);
- тап по цитате ответа — прокрутка к исходному сообщению;
- тап по реакции — поставить/убрать свою реакцию;
- тап по изображению — полноэкранный просмотр;
- ссылки в тексте сообщений кликабельны и открываются в браузере.

При отсутствии сети может показываться плашка с кэшированными сообщениями и кнопкой **Обновить**.

#### Поле ввода

- кнопка **Вложение** — выбор:
  - **Галерея** (фото из приложения);
  - **Камера**;
  - **Файл**;
- многострочное поле текста (до 5 строк);
  - клавиша **Enter** / **Ввод** добавляет новую строку;
  - отправка — только кнопкой **Отправить**;
- превью прикреплённого файла (если есть);
- кнопка **Отправить**;
- **долгое нажатие** на **Отправить** — меню режимов отправки:
  - **Отправить без звука** — получатель увидит сообщение без звука уведомления;
  - **Отложить отправку** — выбор даты и времени (до года вперёд); сообщение появится в чате сразу как отложенное и уйдёт автоматически в назначенное время, даже если вы закроете чат или приложение.

**Режим ответа:** над полем ввода полоска «Ответ на …» с кнопкой закрыть.

**Режим редактирования:** над полем «Редактирование сообщения» и кнопка **Отмена** (только для своих текстовых сообщений).

#### Действия с сообщением (тап по сообщению)

Открывается нижняя панель:

**Быстрые реакции:** 👍 ❤️ 😂 😮 😢 🙏 🔥 👏 и выбор любого эмодзи.

**Действия:**

- **Ответить**
- **Редактировать** (только свои текстовые)
- **Копировать**
- **Переслать**
- **Удалить** (только свои)

#### Панель при выборе одного сообщения

Внизу: **Ответить**, **Переслать**.

#### Если пользователь покинул чат

Вместо поля ввода — текст «Вы покинули чат» и кнопка **Вернуться в чат** (если возврат разрешён).

#### После отправки фото

Если в настройках включено «Предлагать указать, кто на фото», может появиться предложение отметить людей на снимке.

### 6.4. Информация о чате (нижняя панель)

Открывается тапом по названию или аватару в шапке переписки.

#### Шапка панели

- для **личного чата** — крупное фото собеседника (тап — полноэкранный просмотр);
- для **семейного чата** и **чата подготовки поздравления** — фирменная шапка с декоративным фото;
- имя по центру; если задано личное название, показывается в формате **«Киса (Иван)»** — ваше имя и исходное;
- иконка **карандаша** рядом с именем — переименовать чат (диалог с полем ввода, подсказкой «Видно только вам» и кнопкой **Сбросить**, если название уже задано);
- под именем — статус собеседника (в личном чате) или число участников (в группе/семейном);
- меню **⋮** — дополнительные действия (если доступны).

При прокрутке шапка сворачивается: фото уменьшается, появляется компактный аватар.

**Кнопки-действия** (под шапкой, если применимо):

| Кнопка | Что делает |
|--------|------------|
| **Уведомления** | Отключить уведомления: 1 час / 4 часа / 8 часов / 24 часа / Навсегда; или **Включить уведомления** |
| **Вернуться в чат** | Вернуться в чат после выхода (семейный чат) |
| **Покинуть чат** | Выйти из чата (с подтверждением) |
| **Добавить участников** | Добавить родственников в группу (только групповые чаты) |

**Вкладки:**

- **Участники** — в групповых и семейном чатах; список с аватарами, тап → профиль;
- **Галерея** — сетка изображений из чата; тап — открыть; долгое нажатие — перейти к сообщению;
- **Ссылки** — ссылки из сообщений; **Открыть ссылку** / **Перейти к сообщению**.

В личных чатах вкладка **Участники** скрыта — остаются **Галерея** и **Ссылки**.

### 6.5. Пересылка сообщений

Экран **«Переслать»**:

- список чатов с чекбоксами;
- **Выбрать все**;
- **Отправить (N)** — переслать в выбранные чаты.

### 6.6. Аудиозвонки (только личные чаты)

#### Исходящий звонок

Экран **«Звонок: {имя}»**:

- статусы: Подключение… / Звоним… / Разговор идёт;
- **Громкая связь** — переключение динамика (по умолчанию разговорный, у уха);
- **Завершить** (красная кнопка).

При отказе в доступе к микрофону — диалог с **Настройки** / **Повторить**.

**Поведение экрана:** при разговоре через разговорный динамик, если поднести телефон к уху — экран гаснет (как в обычном телефонном звонке). При включённой громкой связи экран остаётся включённым.

#### Входящий звонок

Полноэкранный экран **«Входящий звонок»**:

- аватар, имя, степень родства звонящего;
- **Сбросить** — отклонить;
- **Ответить** — принять и перейти в разговор.

Приходит по push-уведомлению или когда приложение открыто. Системная кнопка «назад» отклоняет звонок.

На **веб-версии и PWA** тап по уведомлению о звонке открывает экран входящего звонка, в том числе если приложение было закрыто.

#### История звонков в переписке

После завершения, пропуска или отклонения звонка в **личном чате** появляется системная плашка в ленте сообщений:

| Ситуация | Что показывается |
|----------|------------------|
| Состоявшийся разговор | «Входящий звонок» / «Исходящий звонок» + длительность (мм:сс) |
| Пропущенный | «Пропущенный входящий» / «Пропущенный исходящий» |
| Отклонённый | «Входящий отклонён» / «Вызов отклонён» |
| Отменённый | «Входящий отменён» / «Исходящий отменён» |

У пропущенного входящего и отклонённого исходящего — кнопка **Перезвонить** (иконка телефона).

История звонков видна обоим участникам личного чата и не попадает в семейную ленту.

## 7. Вкладка «Семья»

Заголовок: **Семья**.

**Кнопка в шапке:** **Добавить в семью** (иконка человека с плюсом) — пригласить нового родственника.

Две вкладки: **Участники** и **Дерево**.

### 7.1. Участники

**Поиск** — поле «Поиск» с кнопкой очистки.

**Фильтры по родству (чипы):**

- **Все**
- **Близкие**
- **Родители**
- **Братья/сестры**
- **Двоюродные**

**Список участников.** Каждая строка:

- аватар;
- имя;
- степень родства относительно вас (Жена, Сын, Брат, Вы, и т.д.);
- день рождения.

Тап по строке:

- на себя — открывается **Профиль** (из раздела «Ещё»);
- на другого — **Профиль участника**.

### 7.2. Дерево

Интерактивная **карта родства** — граф связей между родственниками.

**Шапка карты:**

- ваш аватар и имя;
- подпись «Вы в центре карты» (или «Центр карты», если смотрите от лица другого);
- статистика: сколько человек в семье / на карте / рядом / связей;
- **Ко мне** — вернуть вас в центр дерева (если смотрели от другого человека).

**Узлы дерева:**

- круглый аватар;
- имя;
- подпись родства (Сын, Жена, Брат, Вы, и т.д.).

**Жесты:**

- **Тап** по узлу — профиль участника;
- **Долгое нажатие** — сделать этого человека центром дерева;
- **Щипок** — масштаб;
- **Перетаскивание** — перемещение по карте.

Дети отображаются под парой родителей, с линиями связи от обоих.

Если связей мало — подсказка: «Добавляйте родственников через приглашения — связи появятся здесь».

### 7.3. Приглашение в семью

Диалог **«Пригласить в семью»**:

- выпадающий список **степени родства** (кто приглашаемый по отношению к вам);
- пояснение: ссылка действует 24 часа, одноразовая;
- **Создать ссылку** — открывает системное меню «Поделиться» с текстом приглашения.

Получатель переходит по ссылке → онбординг → уточняющие вопросы о родстве (при необходимости) → попадает в семью.

## 10. Галерея

### 10.1. Меню галереи

Экран **«Галерея»** с вкладками:

- **Мои** — ваша личная галерея;
- **Все** — общая семейная галерея (фото всех участников, к которым у вас есть доступ).

В **Мои** есть плавающая кнопка **Альбом** (создать альбом).

### 10.2. Обзор галереи (вкладки Мои / Все)

**Карточка «Все фото»** — все доступные снимки.

**Группы альбомов** (чипы-заголовки с количеством):

- **Мои альбомы** / **Альбомы** — пользовательские альбомы;
- **Люди** — автоматические подборки по распознанным лицам;
- **Места** — по геолокации;
- **Годы** — по году съёмки.

Сетка альбомов с обложкой и названием.

**Долгое нажатие** на свой альбом:

- **Редактировать**
- **Удалить**

**Кнопка «Альбом»** (FAB) — создать новый альбом (в своей галерее или в семейной — если есть права).

Может показываться подсказка о распознавании лиц на фото.

### 10.3. Создание / редактирование альбома

Диалог **«Новый альбом»** / **«Редактировать альбом»**:

- **Название**
- **Кому доступен:**
  - Всем в семье;
  - Только выбранным людям (список с галочками);
  - Всем, кроме выбранных.
- **Кто может добавлять фото:**
  - Только создатель;
  - Все;
  - Выбранные люди.

Кнопки **Сохранить** / отмена.

### 10.4. Экран альбома

Заголовок — название альбома.

**Шапка:**

- **Поиск** — поиск по тегам;
- фильтр по людям на фото: **Все люди**, **Не определены**, аватары конкретных людей;
- **Выбрать** — режим множественного выбора.

**В режиме выбора:**

- **Выбрать все** / **Снять все**;
- меню: **Скачать**, **Поделиться**, **Добавить тег**, **Удалить** (для своих фото).

**Меню альбома** (если есть права управления):

- **Удалить дубликаты**
- **Редактировать альбом**
- **Удалить альбом**

**Плавающая кнопка «+»** (если можно добавлять фото):

- **Из галереи** (выбор внутри приложения);
- **Галерея телефона**;
- **Файлы с телефона**;
- **Камера**.

Для альбомов, привязанных к событию календаря, может быть баннер автосинхронизации фото с телефона (Android) с кнопкой **Синхронизировать**.

**Сетка фото:** тап — просмотр; в режиме выбора — долгое нажатие для выделения.

### 10.5. Просмотр фото

Полноэкранный просмотр с пролистыванием между снимками.

**Шапка:**

- **Кто на фото**;
- меню:
  - **Указать, кто на фото**
  - **Удалить фото** / **Убрать из моей галереи**
  - **Поделиться / скачать**

**Внизу:**

- лайк (счётчик);
- комментарии (раскрываемый блок);
- полоска людей на фото;
- теги.

Жесты: свайп между фото, щипок для зума.

## 11. Календарь

Экран **«Календарь»** (из «Ещё»).

Вкладки: **События** | **Месяцы**.

Плавающая кнопка **Событие** — создать новое событие.

### 11.1. Вкладка «События»

- переключатель месяца (стрелки влево/вправо и название месяца);
- список событий по дням:
  - дни рождения (иконка торта);
  - пользовательские события;
  - семейные праздники.

Тап:

- на **своё/чужое событие** — редактирование (если вы создатель или участник);
- на **день рождения** — экран дня рождения.

Пустой месяц: «В этом месяце нет событий».

### 11.2. Вкладка «Месяцы»

Календарная сетка на год с выделением дней, где есть события.

- тап по дню — нижняя панель со списком событий этого дня и кнопкой **Добавить событие**;
- тап по событию — редактирование или просмотр (только чтение для чужих праздников);
- тап по дню рождения — экран дня рождения.

### 11.3. Создание / редактирование события

Экран **«Новое событие»** / **«Редактировать событие»**:

- **Название**
- **Дата начала** / **Дата конца**
- **Напоминание:** Без / 15 минут / 30 минут / 1 час / 1 день
- **Участники события** — галочки у членов семьи
- **Создать альбом** — привязать фотоальбом к событию
  - на Android: **Подтягивать фото с телефона** (автосинхронизация)
  - настройки доступа к альбому (как у обычного альбома)
  - **Кто может добавлять фото**

Кнопки **Создать** / **Сохранить**.

При редактировании — удаление события (**Удалить событие?**).

### 11.4. День рождения

Экран **«День рождения»**:

- имя именинника;
- дата;
- переключатель **Не хочу поздравлять** (исключить из чата подготовки поздравления);
- информация о чате подготовки (если создан);
- **Открыть чат подготовки** — групповой чат для координации поздравления.

## 14. Уведомления

### Типы push-уведомлений

| Тип | Когда приходит | Что происходит при тапе |
|-----|----------------|-------------------------|
| Новое сообщение | Сообщение в чате, где вы участник | Открывается этот чат (с прокруткой к сообщению, если указано) |
| Сообщение без звука | Отправитель выбрал «Отправить без звука» | Открывается чат; уведомление приходит без звука |
| Напоминание календаря | Наступило время напоминания | Открывается календарь |
| Входящий звонок | Звонок в личном чате | Полноэкранный экран входящего звонка |

Если вы уже смотрите этот чат, уведомление о новом сообщении в нём не показывается.

На Android звонки идут через отдельный канал «Звонки» с мелодией; сообщения — через канал «Сообщения».

На веб-версии и PWA для звонка приходит одно уведомление (без дублирования); тап открывает экран входящего звонка.

### Статусы присутствия

В личных чатах и профилях отображается один из статусов:

- **Онлайн**
- **Был только что** (до 1 часа назад)
- **Был недавно** (до 4 дней)
- **Был давно**

## 15. Правила видимости и приватности

Это важно для понимания того, **кто что видит**:

### Лента

- **Личные чаты (1-на-1) никогда не попадают в ленту** — ни текст, ни факт переписки.
- Сообщения из **групповых чатов** видны в ленте только **участникам этой группы**.
- Сообщения из **семейного чата** видны всем членам семьи (кто не покинул чат).
- **Фото, лайки и комментарии** в ленте — только если пользователь может видеть это фото в галерее.
- **События альбомов** — только если у пользователя есть доступ к этому альбому.
- События «новый участник», «обновлён профиль», «календарь» — видны всей семье.

### Чаты

- **Семейный чат** — все члены семьи (можно покинуть и вернуться).
- **Личный чат** — только два участника.
- **Группа** — только добавленные участники; создатель/участники могут добавлять новых из семьи.

### Галерея

- У каждого участника своя галерея; другие видят только то, что разрешено настройками приватности и альбомов.
- **Альбомы** имеют настройки «кому доступен» и «кто может добавлять».
- **Общая галерея «Все»** агрегирует только доступные вам фото всех членов семьи.

```
Вход
├── Вход (VK / Яндекс / Google)
├── Политика конфиденциальности / Пользовательское соглашение

    ├── Главная — лента событий
    ├── Чат — список чатов
    │   ├── Новая группа
    │   ├── Переписка
    │   │   ├── Информация о чате
    │   │   ├── Пересылка
    │   │   ├── Исходящий звонок
    │   │   ├── Входящий звонок
    │   │   └── История звонков (плашки в ленте)
    │   └── (поиск по чатам)
    ├── Семья
    │   ├── Участники → Профиль участника
    │   ├── Дерево родства
    │   └── Приглашение в семью
    └── Ещё

```

## 17. Краткий список возможностей

**Общение:** семейный чат, личные сообщения, группы, ответы, редактирование, удаление, пересылка, реакции, вложения (фото, файлы), многострочный ввод, кликабельные ссылки, поиск по сообщениям, отправка без звука, отложенная отправка, переименование чата для себя, отключение уведомлений по чату, выход/возврат в чат.

**Звонки:** аудиозвонки в личных чатах, громкая связь, гашение экрана у уха, входящий экран с ответом/сбросом, история звонков в переписке с перезвоном, входящие звонки из push на веб/PWA.

**Семья:** список с родством и фильтрами, интерактивное дерево, приглашения по ссылке, профили участников.

**Лента:** события чатов, галереи, календаря, участников — с учётом прав доступа.

**Галерея:** личные и общие фото, альбомы с правами, лица/места/годы, лайки, комментарии, теги, дубликаты, шаринг.

**Календарь:** события, дни рождения, праздники, напоминания, альбомы событий, чаты подготовки поздравлений.

**Профиль:** крупный аватар с просмотром, данные, приватность дня рождения (показ года), цвет темы, выход, удаление.

**Платформы:** Android, iPhone, веб-версия и PWA.

**Интеграции:** вход через OAuth (VK, Яндекс, Google), приглашения по deep link, «Поделиться» из других приложений, push-уведомления, юридические документы на сайте.
